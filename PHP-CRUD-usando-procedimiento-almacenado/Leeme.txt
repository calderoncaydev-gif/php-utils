How to run this script

1.Download the zip file
2.Extract the file and copy spcrud folder
3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
4.Open PHPMyAdmin (http://localhost/phpmyadmin)
5.Create a database with name spcruddb
6.Import spcruddb.sql file(given inside the zip package)
7.Run the script http://localhost/spcrud   (frontend)
